Module 7

Object Detection and Recognition

Connect classical recognition pipelines with modern detection systems, from sliding windows and HOG+SVM to YOLO, Faster R-CNN, SSD, and representation learning.

Lecture Focus

Mapped to CO3 from the course outline.

Lecture 18: Classical Object Detection

Sliding windows, scale pyramids, HOG cells and blocks, linear SVMs, non-maximum suppression, and deformable part models.

Lecture 19: Deep Learning-Based Detection

R-CNN family, region proposal networks, anchor boxes, YOLO, SSD, mAP, IoU, FPS, and speed-accuracy trade-offs.

Infographic

Detection pipeline choices.

C

Classical route

Extract HOG features from windows, classify with SVM, then suppress duplicates.

R

Proposal route

Generate candidate regions, classify each region, and refine boxes.

Y

Single-stage route

Predict boxes and classes in one pass for real-time applications.

M

Measure

Evaluate IoU, mAP, FPS, memory, and robustness across scenes.

Scenario Driven Exploration

Campus safety camera

Choose between HOG+SVM and YOLO for pedestrian detection. Explain what changes if the hardware is an edge device.

Retail shelf monitoring

Compare proposal-based and single-stage detectors when small objects overlap, lighting changes, and speed matters.
