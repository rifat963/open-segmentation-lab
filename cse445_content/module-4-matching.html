The robust matching pipeline.

1

Match descriptors

Generate candidate correspondences using the right distance metric.

2

Filter candidates

Apply ratio test or mutual checks to remove weak matches.

3

Run RANSAC

Estimate a model while rejecting outliers that do not agree geometrically.

4

Warp and evaluate

Use the homography for alignment, then inspect inliers, seams, and reprojection error.
