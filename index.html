CSE438

Digital Image Processing

An undergraduate course covering the fundamentals of image processing — from pixel-level operations and spatial filtering to frequency domain analysis, morphological processing, and classical segmentation techniques.

  • Image fundamentals: representation, sampling, histograms
  • Spatial filtering: mean, Gaussian, Laplacian, unsharp masking
  • Frequency domain: DFT, Butterworth and Gaussian filters
  • Morphological operations: erosion, dilation, opening, closing
  • Segmentation: Canny edges, Otsu thresholding, watershed
  • Image restoration: noise models and Wiener filtering

What is Image Segmentation?

Image segmentation is the task of partitioning an image into meaningful pixel-level regions. Instead of predicting only an image label or drawing a bounding box, segmentation asks a model to describe where each visual concept occurs at pixel resolution.

Definition: A segmentation mask is an image-sized annotation where pixel values encode classes, instances, or both. The mask is the same height and width as the input image, and every pixel carries a label.

From boxes to pixels

Object detection draws rectangular bounding boxes — approximate spatial summaries. Segmentation is more precise: it traces the exact shape of each region. This matters for counting, measuring area, detecting fine boundaries, and compositing objects for downstream use.

Why dense prediction is hard

A segmentation model must produce a decision for every one of the millions of pixels in an image. It must understand global scene context while also preserving fine local detail — an encoder-decoder architecture with skip connections is the standard solution.

Where Computer Vision Is Applied

Pixel-level and object-level understanding is essential across medicine, agriculture, autonomous systems, and earth observation.

Medical Imaging

Organ delineation, tumor measurement, cell counting, and lesion tracking in MRI, CT, and pathology slides.

Precision Agriculture

Crop-versus-weed separation, canopy area measurement, disease detection, and yield estimation from drone imagery.

Autonomous Driving

Road, lane, sidewalk, pedestrian, and obstacle labeling to enable real-time planning decisions in self-driving stacks.

Remote Sensing

Land-cover mapping, building and road extraction, flood detection, and deforestation monitoring from satellite imagery.

Semantic vs Instance vs Panoptic Segmentation

These tasks share the goal of pixel-level understanding but answer different questions. Choosing the right task frames the annotation strategy, model architecture, and evaluation metric.

Task Question Answered Output Typical Metrics
Semantic segmentation What class does each pixel belong to? One class label per pixel; same class merges all instances Pixel accuracy, mIoU, Dice
Instance segmentation Which object instance does each foreground pixel belong to? One binary mask per detected object with class and confidence Mask AP, Mask IoU
Panoptic segmentation How can every pixel be assigned both a semantic class and instance identity? Unified scene map with class and instance IDs; no unlabeled pixels Panoptic Quality (PQ)
Key distinction: Semantic segmentation says "these pixels are cars." Instance segmentation says "these pixels are car 1, those pixels are car 2." Panoptic segmentation does both — and also labels background regions like sky and road.

Semantic Segmentation

Pixel-wise classification using encoder-decoder architectures, multiscale context, and semantic metrics. The foundation of all dense prediction tasks.

  • Class label for every pixel (no instance identity)
  • FCN, U-Net, DeepLab, SegFormer, Mask2Former
  • mIoU, Dice score, pixel accuracy
  • PyTorch code templates included
Open tutorial
Object masks

Instance Segmentation

Object proposals, mask heads, real-time pipelines, and instance-level evaluation using Mask AP.

  • Separate binary mask per object
  • Mask R-CNN, YOLACT, YOLO Seg
  • Mask AP and IoU per instance
Open tutorial
Unified scene map

Panoptic Segmentation

Unified scene understanding that assigns every pixel a semantic label and, for countable objects, an instance identity.

  • Things (countable) and stuff (uncountable) classes
  • Complete pixel assignment — no unlabeled pixels
  • Panoptic Quality (PQ) evaluation

Self-Supervised Segmentation

Representation learning, pseudo-mask generation, and low-annotation segmentation workflows using pretrained encoders.

  • Contrastive and masked pretraining
  • Pseudo-mask generation with DINO and SAM
  • Fine-tuning and semi-supervised strategies

Object Detection

Detect and localize multiple objects in a single image using bounding boxes. The bridge between image classification and pixel-level segmentation — covering two-stage detectors, one-stage detectors, anchor-free methods, and transformer-based detectors.

  • Two-stage: Faster R-CNN with RPN and RoI head
  • One-stage: YOLO family, SSD, RetinaNet
  • Transformer: DETR — no anchors, no NMS
  • Evaluation: mAP@0.5, mAP@[0.5:0.95]

Suggested Learning Path

New to image segmentation? This sequence builds concepts in the right order — from pixel labeling to advanced evaluation and paper reading.

1

Semantic Segmentation

Start with pixel-wise classification. Understand encoder-decoder design, cross-entropy loss, and mIoU before moving to instance-level tasks.
2

Evaluation Metrics

Learn IoU, Dice, pixel accuracy, and mIoU. Use the interactive calculators to build intuition before reading benchmark papers.
3

Object Detection

Study bounding box prediction, anchor-based vs anchor-free approaches, and mAP evaluation. Core topic in CSE445 Computer Vision.
4

Instance Segmentation

Move to per-object masks. Study Mask R-CNN's proposal pipeline, RoIAlign, and Mask AP evaluation.
5

Panoptic Segmentation

Combine semantic and instance understanding. Learn about things vs. stuff and the Panoptic Quality metric.
6

Key Research Papers

Read FCN, U-Net, Mask R-CNN, SegFormer, and Mask2Former. The reading list includes summaries and what to focus on in each paper.

Featured Papers

A short reading path from end-to-end dense prediction to instance masks and universal segmentation models. Each paper introduced a lasting architectural idea still referenced today.

2015 · Semantic segmentation

Fully Convolutional Networks for Semantic Segmentation

Introduced end-to-end pixel-level prediction by converting classification CNNs into fully convolutional networks. Established upsampling, skip connections, and dense labeling as the core recipe.

Read paper

2017 · Instance segmentation

Mask R-CNN

Extended Faster R-CNN with a parallel mask branch and RoIAlign for precise spatial alignment. Made instance-level masks practical and set the Mask AP evaluation standard.

Read paper

2022 · Universal segmentation

Mask2Former

Unified semantic, instance, and panoptic segmentation using masked attention transformers and a shared mask classification formulation — one model, three tasks.

Read paper
