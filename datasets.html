Segmentation Datasets

Dataset choice affects label format, model architecture, evaluation metric, and project scope. These cards summarize common options for student projects and literature review.

Cityscapes

Urban street-scene dataset for road understanding and autonomous driving research.

Task type: Semantic and instance-aware urban scene segmentation.
Annotation format: Fine pixel labels and polygon annotations for selected classes.
Suitable models: DeepLab, HRNet, SegFormer, Mask2Former, panoptic models.
Notes for students: Strong for studying class imbalance, thin structures, and urban "things vs stuff" labels.

COCO

Large-scale natural image dataset with object detection, instance segmentation, captions, and panoptic annotations.

Task type: Instance and panoptic segmentation.
Annotation format: Polygons, compressed masks, categories, image metadata.
Suitable models: Mask R-CNN, YOLACT, YOLO segmentation, DETR-style models, Mask2Former.
Notes for students: Excellent for learning standard benchmarks, but training from scratch can be computationally expensive.

ADE20K

Scene parsing dataset with a broad vocabulary of object and stuff categories.

Task type: Semantic segmentation and scene parsing.
Annotation format: Pixel-level category masks.
Suitable models: UperNet, SegFormer, DeepLab variants, Mask2Former.
Notes for students: Useful for studying many-class segmentation and indoor/outdoor scene diversity.

Pascal VOC

Classic benchmark for object recognition and segmentation with a manageable number of classes.

Task type: Semantic segmentation.
Annotation format: Pixel masks for foreground object categories.
Suitable models: FCN, U-Net-style baselines, DeepLab, lightweight student models.
Notes for students: Good first benchmark because the class list is smaller and many tutorials use it.

Medical Imaging Datasets

Includes organ, lesion, cell, vessel, tumor, and tissue segmentation datasets across modalities.

Task type: Semantic, instance, or volumetric segmentation depending on the dataset.
Annotation format: Binary masks, multiclass masks, 3D volumes, or contours.
Suitable models: U-Net, nnU-Net, attention U-Net, transformer U-Net variants.
Notes for students: Pay attention to patient-level splits, privacy, modality differences, and class imbalance.

Agricultural Disease Segmentation Datasets

Datasets for segmenting diseased leaf regions, crops, weeds, fruits, canopy, and field objects.

Task type: Semantic or instance segmentation for plant regions and disease symptoms.
Annotation format: Pixel masks, polygons, or bounding boxes converted to weak masks.
Suitable models: U-Net, DeepLabV3, SegFormer, YOLO segmentation, SAM-assisted annotation workflows.
Notes for students: Watch for lighting variation, small lesions, background clutter, and dataset bias across crop species.
