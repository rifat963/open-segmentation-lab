Segmentation Datasets

Dataset choice affects annotation format, model architecture, evaluation metric, and project scope. These cards summarize common options for student projects and literature review, with guidance on which tasks each dataset best supports.

Dataset Details

Cityscapes

Urban street-scene dataset recorded from a moving vehicle across 50 European cities. High-quality pixel annotations for 30 classes with 19 used in evaluation.

Task type: Semantic segmentation; instance-aware annotations available for 8 object classes.
Annotation format: Fine pixel labels (5 000 images) and coarse labels (20 000 images). Polygon annotations for instance classes.
Suitable models: DeepLab, HRNet, SegFormer, Mask2Former, panoptic models.
Key challenge: Class imbalance between road/sky and rare pedestrian classes; thin structures like poles; varying lighting.
Notes for students: Strong for studying class imbalance, boundary quality, and urban things-vs-stuff separation.
Semantic Instance Urban Fine-grained labels

COCO

Large-scale natural image dataset with detection, instance segmentation, captions, keypoints, and panoptic annotations. The de facto standard for instance segmentation benchmarks.

Task type: Object detection, instance segmentation, panoptic segmentation, and image captioning.
Annotation format: Polygons and compressed RLE masks per instance. Panoptic annotations cover all pixels including stuff regions.
Suitable models: Mask R-CNN, YOLACT, YOLO segmentation, DETR, Mask2Former.
Key challenge: Scale, occlusion, small objects, and many-class diversity. Training from scratch is computationally expensive.
Notes for students: Use COCO-pretrained weights and fine-tune on your domain. Study pycocotools for annotation handling and Mask AP computation.
Instance Panoptic Large scale Benchmark standard

ADE20K

Scene parsing dataset covering 150 semantic categories across indoor and outdoor environments. Used in SegFormer, Swin, and Mask2Former benchmarks.

Task type: Semantic segmentation and scene parsing. Some annotations also support instance-level labels.
Annotation format: Pixel-level category masks. Additional object instance masks and object part annotations.
Suitable models: UperNet, SegFormer, DeepLab variants, Mask2Former.
Key challenge: 150 categories with significant class imbalance and fine-grained distinctions (e.g., different wall types).
Notes for students: Useful for studying many-class semantic segmentation and evaluating transformer-based models on a diverse scene vocabulary.
Semantic 150 classes Indoor + outdoor

Pascal VOC

Classic benchmark for object recognition and segmentation. 20 foreground object classes plus background. The first widely-used semantic segmentation standard.

Task type: Semantic segmentation (VOC 2012) and object detection.
Annotation format: Pixel masks for 20 foreground object categories. Boundary annotations available with SBD extension.
Suitable models: FCN, U-Net-style baselines, DeepLab, lightweight student models.
Key challenge: Small dataset by modern standards. Augmentation and pretrained weights are essential for good performance.
Notes for students: The best first benchmark. Many tutorials and pretrained models use VOC. Class list is manageable for learning annotation workflows.
Semantic Beginner friendly 20 classes

Medical Imaging Datasets

Includes organ delineation, lesion detection, cell counting, vessel tracking, and tissue segmentation datasets across CT, MRI, and pathology slide modalities.

Task type: Semantic, instance, or volumetric (3D) segmentation depending on the dataset and modality.
Annotation format: Binary masks, multiclass masks, 3D volume labels (NIfTI, DICOM), or contour files.
Notable datasets: ISIC (skin lesion), BraTS (brain tumor), LUNA (lung nodule), MoNuSeg (nuclei), CHAOS (abdominal organs).
Suitable models: U-Net, nnU-Net, Attention U-Net, transformer U-Net variants, MedSAM.
Notes for students: Observe patient-level train/val/test splits to prevent data leakage. Account for class imbalance, modality differences, and imaging artifacts. Privacy constraints limit dataset sharing in some cases.
Medical Semantic 3D volumes available Patient splits required

Agricultural Disease Segmentation

Datasets for segmenting diseased leaf regions, crops, weeds, fruits, canopy cover, and field-level objects from drone, ground-level, or close-up imagery.

Task type: Semantic or instance segmentation for plant regions, disease symptoms, and crop rows.
Annotation format: Pixel masks, polygon annotations, or bounding boxes converted to weak masks. Some use SAM-assisted annotation workflows.
Notable datasets: PlantVillage (leaf disease), DeepWeeds, iBean, GrowliFlower, Crop and Weed Field Image Dataset.
Suitable models: U-Net, DeepLabV3, SegFormer, YOLO segmentation, SAM with domain fine-tuning.
Notes for students: Watch for lighting variation, small lesion sizes, cluttered backgrounds, and severe domain shift between crop species and imaging conditions. Always inspect annotation quality before training.
Agriculture Semantic Small scale Domain shift risk
Choosing a dataset for a student project: Start with Pascal VOC or a small medical dataset. Both have manageable sizes, clear annotation formats, and existing baseline results to compare against. Scale up to COCO or Cityscapes once training and evaluation pipelines are working correctly.
Data quality warning: SAM-generated pseudo masks and crowd-sourced annotations may contain noise. Always visually inspect a random sample before training. Poor annotation quality is a common source of unexpectedly low performance that is easily misdiagnosed as a model problem.
