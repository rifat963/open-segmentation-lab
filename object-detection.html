Computer Vision

Object Detection

Detect and localize multiple objects in an image using bounding boxes — the bridge between image classification and pixel-level segmentation.

What is Object Detection?

Object detection combines localization and classification: given an image, predict what objects are present and where each one is, expressed as an axis-aligned bounding box with a class label and confidence score.

Definition: An object detector outputs a set of tuples (class, confidence, x, y, w, h) — one per detected object. Unlike segmentation, the spatial output is a rectangle, not a pixel mask.

Two-stage detectors

Generate region proposals first, then classify each proposal. Examples: R-CNN, Fast R-CNN, Faster R-CNN. Higher accuracy at the cost of slower inference.

One-stage detectors

Predict boxes and class scores directly from the full image in a single forward pass. Examples: YOLO family, SSD, RetinaNet. Faster inference, suitable for real-time use.

Key Model Families

Model Year Approach Strength
Faster R-CNN 2015 Two-stage, RPN + RoI head High accuracy, flexible backbone
YOLOv5 / YOLOv8 2020–2023 One-stage, anchor-based / anchor-free Real-time, easy to deploy
RetinaNet 2017 One-stage, focal loss Handles class imbalance well
DETR 2020 Transformer encoder-decoder No hand-crafted anchors or NMS

Evaluation: mAP

Mean Average Precision (mAP) is the standard detection metric. It averages the area under the precision–recall curve across all classes and IoU thresholds.

IoU threshold: A predicted box is a true positive only if its overlap with the ground-truth box exceeds a threshold (e.g., 0.5 for PASCAL VOC, 0.5–0.95 for COCO). Lower thresholds are more lenient on localization.

PASCAL VOC metric

mAP@0.5 — a single IoU threshold of 0.5. Easier to achieve but less sensitive to precise localization.

COCO metric

mAP@[0.5:0.95] — averaged over ten IoU thresholds from 0.5 to 0.95 in steps of 0.05. Rewards tighter boxes.

Object Detection vs Segmentation

Task Spatial Output Precision Typical Use Case
Object detection Bounding box (x, y, w, h) Rectangle approximation Counting, coarse localization, real-time systems
Instance segmentation Pixel-level binary mask per object Exact object boundary Shape analysis, compositing, medical imaging
