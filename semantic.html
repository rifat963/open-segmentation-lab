Semantic Segmentation

Semantic segmentation assigns a category label to every pixel. It is the foundation for many dense prediction tasks in medical imaging, remote sensing, autonomous navigation, and agricultural analysis.

Semantic Segmentation Definition

Semantic segmentation is the computer vision task of assigning a semantic class label to every pixel in an image. If an image has height H, width W, and C target classes, a segmentation model usually predicts a score tensor with shape C x H x W. The final class map is obtained by choosing the highest-scoring class at each pixel.

Logit tensor Z = fθ(X),   Z ∈ RC × H × W Pixel prediction ŷi,j = argmaxc Zc,i,j
Definition: Semantic segmentation treats all pixels of the same class as the same category. Two cars, two crop leaves, or two tumor regions share one class label unless the task is changed to instance segmentation.

How Dense Prediction Works

A semantic segmentation network keeps spatial structure throughout the model. The encoder extracts visual features, the decoder restores spatial resolution, and the classifier projects features into class logits. The model learns boundaries, textures, shapes, and context because the loss is computed at the pixel level.

Pixel-wise cross entropy L = -1/(H W) Σi Σj log softmax(Z:,i,j)yi,j

Encoder

Extracts low-level edges, textures, high-level parts, and global context using CNN or transformer layers.

Decoder

Upsamples feature maps and combines coarse semantic context with fine spatial detail.

Classifier Head

Uses a small convolutional layer to convert feature channels into class logits for every pixel.

Loss and Metrics

Optimizes pixel-wise classification loss and evaluates overlap with mIoU, Dice, or pixel accuracy.

Important Model Families

FCN showed that classification networks could be converted into dense predictors. U-Net added an encoder-decoder structure with skip connections, making it especially strong for medical and agricultural imagery. DeepLab uses atrous convolution and multiscale context. Modern transformer-based models such as SegFormer and Mask2Former improve representation learning and support strong performance across complex scenes.

Exam note: U-Net is a strong baseline when labeled data is limited and boundary localization matters. DeepLab is useful for discussing multiscale context, while SegFormer and Mask2Former are useful for transformer-era segmentation.

Applications

Medical Imaging

Segment organs, tumors, lesions, cells, vessels, or tissue regions for measurement and clinical research.

Agriculture

Separate crop, weed, soil, canopy, leaf disease, and plant stress regions for precision agriculture.

Autonomous Driving

Estimate road, lane, sidewalk, vegetation, vehicle, pedestrian, and traffic-scene regions.

Remote Sensing

Map buildings, roads, water, forest, and land-cover classes from aerial or satellite images.

Basic Dataset and Metric Utilities

Libraries used: PyTorch for tensors and datasets. This tab prepares class-index masks and computes mIoU from logits.

import torch
from torch.utils.data import Dataset

class SegmentationDataset(Dataset):
    # images: list/tensor of RGB images with shape 3 x H x W
    # masks: list/tensor of class-index masks with shape H x W
    def __init__(self, images, masks, image_transform=None):
        self.images = images
        self.masks = masks
        self.image_transform = image_transform

    def __len__(self):
        return len(self.images)

    def __getitem__(self, index):
        image = self.images[index].float()
        mask = self.masks[index].long()

        if self.image_transform:
            image = self.image_transform(image)

        return image, mask

def mean_iou(logits, masks, num_classes):
    # logits: N x C x H x W, masks: N x H x W
    predictions = torch.argmax(logits, dim=1)
    scores = []

    for cls in range(num_classes):
        pred_cls = predictions == cls
        mask_cls = masks == cls
        intersection = torch.logical_and(pred_cls, mask_cls).sum()
        union = torch.logical_or(pred_cls, mask_cls).sum()

        if union > 0:
            scores.append(intersection.float() / union.float())

    return torch.stack(scores).mean()

Explanation: Semantic segmentation masks must store class ids, not RGB colors. nn.CrossEntropyLoss expects logits with shape N x C x H x W and masks with shape N x H x W.

U-Net Style Model

Libraries used: PyTorch torch.nn for convolution blocks, pooling, upsampling, and skip connections.

import torch
import torch.nn as nn


class DoubleConv(nn.Module):
    # Two convolution layers are used in many U-Net implementations.
    def __init__(self, in_channels, out_channels):
        super().__init__()
        self.block = nn.Sequential(
            nn.Conv2d(in_channels, out_channels, kernel_size=3, padding=1),
            nn.BatchNorm2d(out_channels),
            nn.ReLU(inplace=True),
            nn.Conv2d(out_channels, out_channels, kernel_size=3, padding=1),
            nn.BatchNorm2d(out_channels),
            nn.ReLU(inplace=True)
        )

    def forward(self, x):
        return self.block(x)


class MiniUNet(nn.Module):
    def __init__(self, num_classes):
        super().__init__()
        self.enc1 = DoubleConv(3, 32)
        self.enc2 = DoubleConv(32, 64)
        self.pool = nn.MaxPool2d(2)

        self.bridge = DoubleConv(64, 128)

        # ConvTranspose2d upsamples the low-resolution feature map.
        self.up2 = nn.ConvTranspose2d(128, 64, kernel_size=2, stride=2)
        self.dec2 = DoubleConv(128, 64)
        self.up1 = nn.ConvTranspose2d(64, 32, kernel_size=2, stride=2)
        self.dec1 = DoubleConv(64, 32)

        # 1 x 1 convolution maps features to class logits.
        self.classifier = nn.Conv2d(32, num_classes, kernel_size=1)

    def forward(self, x):
        skip1 = self.enc1(x)
        skip2 = self.enc2(self.pool(skip1))
        x = self.bridge(self.pool(skip2))

        # Concatenate decoder features with encoder skip features.
        x = self.up2(x)
        x = torch.cat([x, skip2], dim=1)
        x = self.dec2(x)

        x = self.up1(x)
        x = torch.cat([x, skip1], dim=1)
        x = self.dec1(x)

        return self.classifier(x)

Explanation: U-Net preserves boundary information through skip connections. The decoder uses encoder features to recover spatial detail lost during pooling.

DeepLabV3 with Torchvision

Libraries used: PyTorch and Torchvision. Torchvision provides pretrained DeepLabV3 models with ResNet backbones.

import torch
from torchvision.models.segmentation import deeplabv3_resnet50


def build_deeplabv3(num_classes):
    # weights=None avoids downloading pretrained weights.
    # Use pretrained weights only when your environment permits downloads.
    model = deeplabv3_resnet50(weights=None, num_classes=num_classes)
    return model


model = build_deeplabv3(num_classes=4)
images = torch.randn(2, 3, 256, 256)

# Torchvision segmentation models return a dictionary.
outputs = model(images)
logits = outputs["out"]          # shape: N x C x H x W
pred_masks = logits.argmax(dim=1)

Explanation: DeepLabV3 uses atrous convolution and multiscale context. In Torchvision, the main semantic logits are stored in outputs["out"].

SegFormer with Hugging Face Transformers

Libraries used: PyTorch and Hugging Face Transformers. SegFormer combines a hierarchical transformer encoder with a lightweight decoder.

import torch
import torch.nn.functional as F
from transformers import SegformerForSemanticSegmentation


num_classes = 4
model = SegformerForSemanticSegmentation.from_pretrained(
    "nvidia/segformer-b0-finetuned-ade-512-512",
    num_labels=num_classes,
    ignore_mismatched_sizes=True
)

images = torch.randn(2, 3, 512, 512)
outputs = model(pixel_values=images)

# SegFormer logits may be lower resolution than the input image.
logits = F.interpolate(
    outputs.logits,
    size=images.shape[-2:],
    mode="bilinear",
    align_corners=False
)
pred_masks = logits.argmax(dim=1)

Explanation: Transformer segmentation models often output lower-resolution logits. Upsampling makes predictions align with the original mask size for loss and evaluation.

Mask2Former-style Inference

Libraries used: PyTorch and Hugging Face Transformers. Mask2Former predicts masks and class scores using a mask classification formulation.

import torch
from transformers import Mask2FormerForUniversalSegmentation


model = Mask2FormerForUniversalSegmentation.from_pretrained(
    "facebook/mask2former-swin-tiny-ade-semantic"
)

images = torch.randn(1, 3, 512, 512)
outputs = model(pixel_values=images)

# Class scores for predicted masks.
class_logits = outputs.class_queries_logits

# Soft mask logits for each predicted mask query.
mask_logits = outputs.masks_queries_logits

# In production, use the matching image processor post-processing
# utilities to convert query masks into a semantic map.
print(class_logits.shape, mask_logits.shape)

Explanation: Mask2Former differs from ordinary per-pixel classifiers. It predicts a set of mask queries and class scores, then post-processing converts those predictions into semantic, instance, or panoptic outputs.

Training and Inference Template

Libraries used: PyTorch for training, torch.optim for optimization, nn.CrossEntropyLoss for pixel-wise supervised learning, and torch.no_grad for inference.

import torch
import torch.nn as nn


device = "cuda" if torch.cuda.is_available() else "cpu"
num_classes = 4

# Replace MiniUNet with DeepLabV3, SegFormer, or another model.
model = MiniUNet(num_classes=num_classes).to(device)
criterion = nn.CrossEntropyLoss()
optimizer = torch.optim.Adam(model.parameters(), lr=1e-3)


def train_one_epoch(model, train_loader):
    model.train()
    total_loss = 0.0

    for images, masks in train_loader:
        images = images.to(device)
        masks = masks.to(device)

        logits = model(images)
        loss = criterion(logits, masks)

        optimizer.zero_grad()
        loss.backward()
        optimizer.step()

        total_loss += loss.item()

    return total_loss / len(train_loader)


@torch.no_grad()
def predict_mask(model, image):
    model.eval()
    image = image.unsqueeze(0).to(device)    # 1 x 3 x H x W
    logits = model(image)
    prediction = logits.argmax(dim=1)        # 1 x H x W
    return prediction.squeeze(0).cpu()
Cross entropy per pixel Li,j = -log p(yi,j | X, i, j)

Explanation: During training, use raw logits and class-index masks. During inference, use argmax to convert logits into a discrete semantic mask.
