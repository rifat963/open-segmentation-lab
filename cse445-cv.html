Courses

CSE445 Computer Vision

An advanced undergraduate course on deep learning for visual understanding — covering classification, detection, segmentation, and self-supervised representation learning.

Course Overview

CSE445 builds on the fundamentals of image processing to study how modern neural networks interpret visual data. Students learn to design, train, and evaluate deep learning pipelines for real-world vision tasks.

Course Code

CSE445 — Computer Vision
Department of Computer Science and Engineering
East West University, Dhaka

Prerequisites

CSE438 Digital Image Processing, introductory deep learning or machine learning, Python with NumPy and PyTorch.

Topics Covered

Module 1 — CNN Foundations

  • Convolution, pooling, and receptive fields
  • AlexNet, VGG, ResNet, EfficientNet architectures
  • Transfer learning and fine-tuning strategies

Module 2 — Image Classification

  • Softmax classifier and cross-entropy loss
  • Data augmentation and regularization
  • Top-1 and Top-5 accuracy on ImageNet

Module 3 — Object Detection

  • Region proposal networks and RoI pooling
  • Anchor-free detection: FCOS, CenterNet
  • mAP evaluation and NMS

Module 4 — Image Segmentation

  • Semantic segmentation: FCN, DeepLab, SegFormer
  • Instance segmentation: Mask R-CNN, YOLACT
  • Panoptic segmentation and Panoptic Quality

Module 5 — Vision Transformers

  • Attention mechanism and self-attention
  • ViT, DeiT, Swin Transformer
  • Hybrid CNN-Transformer designs

Module 6 — Self-Supervised Learning

  • Contrastive learning: SimCLR, MoCo
  • Masked image modeling: MAE, BEiT
  • DINO and DINOv2 for dense prediction

Recommended Resources

Primary References

Goodfellow, I., Bengio, Y., & Courville, A. (2016). Deep Learning. MIT Press. Supplemented by course lecture notes and selected ArXiv papers.

Lab Environment

PyTorch with torchvision. Lab assignments use COCO and Cityscapes datasets. Students are expected to use Google Colab or a local GPU.
