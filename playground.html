Interactive tools

Segmentation Playground

These tools run entirely in the browser with vanilla JavaScript. Use the simulator to compare segmentation task outputs, then practice metric calculations with small classroom examples.

Segmentation Task Simulator

Switch between semantic, instance, and panoptic segmentation to see how the same scene is represented differently.

Semantic mode shows one class color per pixel.

Simulation Controls

Task mode

Semantic View

Pixels with the same category share one color, even when they belong to different objects.

Instance View

Each countable object receives its own mask and identity, while background stuff is not the main output.

Panoptic View

Stuff regions and thing instances appear together in one complete, non-overlapping scene map.
